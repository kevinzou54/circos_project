Description


This is a user-friendly online tool to generate circular data visualization for gene
rearrangements. Simply upload your file, select some parameters, such as radius of the
Circos and colors of the link, and click onÂ Submit, without the need in downloading or
installing the Circos software package.


The Circos image generated is outputted as a .png and/or .SVG and then be downloaded.


Radius:
Radius range (0-1)




Link Radius:
The link radius controls the the radius of the inner plot displaying the links.
The link radius entered will be scaled to the radius of the entire plot
For example, if the radius of the entire plot is 0.3, and you enter 0.5 for
the link radius, then the link radius will be 0.15

Karyotype - Ideogram
Homo sapiens (GRCh37/hg19)
For reference instead of using chr# to specify chromosomes, use hs#
Format: hs1,hs6,hs9
By default, the entire karyotype will be displayed. If you only want specific chromosomes please enter them here:
This would display chromosomes 1, 6, and 9

Link Data:
Link format: hsX coord1 coord1 hsY coord2 coord2 color=clr

Breakdown of format:

Gene 1 coordinates Gene 2 coordinates Line color
Chromosome # start end Chromosome # start end
hs1 10,000 11,000 hs6 1,000 1,000
